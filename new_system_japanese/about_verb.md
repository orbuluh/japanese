# どうし(動詞)

## 動詞分類


- 五段活用動詞區分三類:
  - G-1,
  - G-2 (a.k.a. 上一段/下一段動詞)
  - 不規則動詞

### 不規則動詞只有兩個: くる(來)　する(做)

|        | する (做)(不規則) | くる (來)(不規則) |
|--------|-----------------|-----------------|
| ない-形  | `し` `ない`        | `こ` `ない`        |
| ます-形  | `し` `ます`        | `き` `ます`        |
| 基本形 | する              | くる              |
| 假定形 | す`れば`           | く`れば`           |
| 意志形 | `し` `よう`        | `こ` `よう`        |
| て-形   | `し` `て`         | `き` `て`         |


- G-1, G-2的活用型變化不同

### G-1

- G-1中 ない形 不加 ない 的部分稱之為 "ない形詞幹"
- G-1中 ます形 不加 ます 的部分稱之為 "ます形詞幹"
- 所有G-1動詞都有自己的活用形
- 各種形式裡的 ない ます ば　う　對於G-1是通用

|        | いく (去)   | いらっしゃる(來/去 to go) | おる(有)      | ある(有)          |
|--------|-----------|---------------------|-------------|-----------------|
| ない-形  | い`か` `ない` | いらっしゃ`らない`          | お`らない` (*2) | ~~あら~~`ない` (*3) |
| ます-形  | い`き` `ます` | いらっしゃ`います` (*1)     | お`ります`      | あ`ります`          |
| 基本形 | い`く`      | いらっしゃ`る`            | お`る`        | あ`る`            |
| 假定形 | い`け` `ば`  | いらっしゃ`れば`           | お`れば`       | あ`れば`           |
| 意志形 | い`こ` `う`  | いらっしゃ`るう`           | お`ろう` (*2)  | あ`ろう`           |
| て-形   | い`って`     | いらっしゃ`って`           | お`って` (*2)  | あ`って`           |

- いる (G-2)　いらっしゃる (G-1)　おる (G-1)　ある (G-1) 表示存在 在~ 位於~ 有~ 等意思
- `(*1)` いらっしゃる 的　ます　形是 いらっしゃ`い`ます　不是　いらっしゃ`り`ます
- `(*2)` おる` 的 ない形 意志形 和 て形 很少用
- `(*3)` ある　的ない形 是 ない　而不是　おらない

- 和上級或不熟悉的人說話 或是在鄭重場合 應改用　いらっしゃる　或 おる　取代 いる
  - いらっしゃる 用於聽話人 聽話人的家屬 與聽話人相關的人 以及上司
    - おどうさんは　いらっしゃいますか。 你父親在嗎
  - おる　用於說話人 說話人的家屬 與說話人相關的人
    - すみません、いま　おりません。對不起我爸現在不在
- ある 用於無生命物 和植物
  - パンが　ないんです。(沒有麵包了)
  - おそばは　どこに　おるんですか。 (蕎麥麵在哪)
- いる 用於有生命物(不包括植物) 以及有駕駛員的火車 汽車 船 飛機 (G-2變化型看下面)

### G-2

- G-2動詞各種形式中的 ない/ます 去除之後即為"G-2動詞詞幹"
- G-2動詞詞幹(基本形去掉る)從不變化。
  - 每一種動詞活用只需要在詞幹後面加上相對應的字

|        | おきる (起床) | いる (做) |
|--------|------------|---------|
| ない-形  | おき`ない`     | い `ない`  |
| ます-形  | あき`ます`     | い `ます`  |
| 基本形 | あき`る`      | い`る`    |
| 假定形 | あき`れば`     | い`れば`   |
| 意志形 | あき`よう`     | い`よう`   |
| て-形   | あき`て`      | い`て`    |
















### Ｇ-１六種基本活用形公式

- :one: **ない-形　＋んです**
- :two: **ます-形　...ます/ません**
- :three: **基本形　＋んです**
- :four: **假定形　＋いいですか : 詢問別人的意見**
  - *なんじに　いけあば　いいですっか。(幾點去好呢)*
    - なんじに (nan-ji-ni) - what time
    - いけば (i-ke-ba) - should go (conditional form of the verb "iku")
    - いいですか (ii-desu-ka) - is it good? (polite way to ask a question)
  - *いつ　いけば　いいですか。(甚麼時候去好呢)*
    - いつ (i-tsu) - when
    - いけば (i-ke-ba) - should go (conditional form of the verb "iku")
    - いいですか (ii-de-su-ka) - is good? (polite form of "ii desu ka")
  - *なんようびに　くれば　いいですか。(星期幾來好呢)*
    - なんようびに (nan-yo-u-bi-ni) - what day
    - くれば (ku-re-ba) - should come (conditional form of the verb "kuru")
    - いいですか (ii-desu-ka) - good? (polite question marker)
- :five: **意志形　＋とおもいます : 說話者的打算/想法/主意**
  - *ブールへ　いこうと　おもいます。(我想去泳池)*
    - ブール (bu-ru) - pool
    - へ (e) - particle indicating direction
    - いこうと (i-ko-u-to) - thinking of going (volitional form of the verb "iku" combined with the particle "to" for expressing intention)
    - おもいます (o-moi-masu) - thinking (present tense of the verb "omou")
  - *げつようびに　いこうと　おもいます。(我想在星期一去)*
    - げつようび (getsu-yo-u-bi) - Monday
    - に (ni) - particle indicating time or destination
    - いこう (i-ko-u) - go (volitional form of the verb "iku")
    - と (to) - if, when, that (conjunction particle)
    - おもいます (o-mo-i-masu) - think (present tense of the verb "omou")
  - *げつようびに　こようと　おまいます。(我想在星期一來)*
    - げつようびに (getsu-yo-u-bi-ni) - on Monday
    - こようと (ko-you-to) - thinking of coming (verb "kuru" in the volitional form "ko" combined with "omou" meaning "to think")
    - おもいます (o-moi-masu) - present/future tense of the verb "omou" meaning "to think"
- :six: **て-形　＋ください : 要求/建議**
  - *わたしの　うちへ　きてください (請來我家)*
    - わたし (wa-ta-shi) - I, me
    - (no) - possessive particle
    - うち (u-chi) - house, home
    - へ (e) - direction particle
    - きてください (ki-te-ku-da-sai) - please come (te-form of the verb "kuru" combined with the polite request form "kudasai")
  - *じゆう　いちじに　きてください (請11點來)*
    - じゆう (ji-yuu) - free, unrestricted
    - いちじ (i-chi-ji) - 1 o'clock
    - に (ni) - at (particle indicating time)
    - きてください (ki-te-kudasai) - please come (te-form of the verb "kuru" combined with "kudasai" which means "please give")
  - *いま　いってください (請現在就去)*
    - いま (i-ma) - now
    - 行ってください (i-tte-ku-da-sai) - please go (te-form of the verb "iku" combined with the imperative form of "kudasai," meaning "please give me")

## 動詞的可能形/可能動詞　（かのうどうじ）

| 動詞基本形 | いく(G-1) | おきる (G-2) | くる     | する    |
|-------|---------|-----------|--------|-------|
| 動詞可能形 | い`ける`   | おき`られる`   | `こられる` | `できる` |

### 活用規則

**G-1動詞**

- 可能形 是 用活用形的第四段假名(`え`段)再加上`る`所構成
  - い`く`　-> い`け`る　-> (第四段: かき`く` `け`こ)
  - よ`む`　-> よ`め`む　-> (第四段: まみ`む` `め`も)

**G-2動詞**

- 可能形 是 用動詞詞幹(基本形去掉る)後添加`られる`構成
  - `おき`る　-> `おき`られる
  - `ね`る　-> `ね`られる

**不規則動詞**

- `くる`　-> `こられる`
- `する`　-> `できる`

### 例外: いける(能去)

いける這樣的動詞可能形不同於他的原來動詞いく(去) 可能態均可視為G-2動詞

| ない-形  | いけ`ない` |
|-------|--------|
| ます-形  | いけ`ます` |
| 基本形 | いける    |
| 假定形 | いけ`れば` |
| 意志形 | いけ`よう` |
| て-形   | いけ`て`  |

### 用動詞可能態時，用が取代を

- にほんりょうり`を`　たべ`る`。 (我吃日本料理)
- にほんりょうり`か`　たべ`られる`。 (我能吃日本料理)

### 動詞可能態的使用(可能/能力)

- にほんごが　はなせる/はなせます。(我會說日語)
- しちじに　こられますか。(妳七點鐘能來嗎)

### 例外： `きく`、`みる`有兩個可能態動詞

**きく -> きける / きこえる**

- きこえる: 意味著 能夠聽到(因為足夠響亮)。其他情況用きける
  - きこえますか。 (你能聽到我的聲音嗎?)
    - きこえます (ki-ko-e-ma-su) - can hear
    - か (ka) - question particle
  - この　ラジオは　がいこくの　ほうそうが　きけます。 (這台收音機能夠收聽到外國廣播電台)
    - この (ko-no) - this
    - ラジオ (ra-ji-o) - radio
    - は (wa) - topic marker particle
    - 外国の (ga-i-ko-ku-no) - foreign
    - 放送が (hou-sou-ga) - broadcast + subject marker particle
    - 聞けます (ki-ke-ma-su) - can listen

**みる -> みられる / みえる**

- みえる: 意味著 能夠看到(因為足夠明亮，足夠大，足夠清晰，足夠近)。其他情況用みられる
  - よく　みえます。(能夠看得清楚)
    - よく (yo-ku) - well, clearly
    - みえます (mi-e-ma-su) - can see
  - この　えいかは　こどもくは　みられません。 (兒童不宜看這部電影)
    - この (ko-no) - this
    - えいか (e-i-ka) - movie
    - は (wa) - topic marker particle
    - こども (ko-do-mo) - children
    - くは (ku-wa) - negation particle indicating "cannot"
    - みられません (mi-ra-re-ma-se-n) - cannot watch

## "する-動詞"

- 一些名詞可以透過加`(を)する`變成動詞
- でんわ　(電話)　:twisted_rightwards_arrows: でんわ`(を)する`  (打電話)
- べんきょう (學習) :twisted_rightwards_arrows: べんきょう`(を)する`  (學)

## ない形 + でください 表示: 否定的要求/建議

- うみでは　およがないでください (請不要在海裡游泳)　(Lesson 4)

## ます-形詞幹 + ましょう 表示: 讓我們...?/要不...?

- いっしょに　やりましょう (我們一起做吧)
- うえのえきで　あいましょう (要不上野車站見吧)

## ます-形的否定形+か　表示: 邀請

- いっしょに　いきませんか (我們一起去好嗎?)

## で形 + いる/いない

- どこに　すんで`いる`んですか。(你住在哪裡)


## 助動詞 だ/です

- だ/です 表示對人、事、物或其狀態的斷定。
- コーさんは　がくせいだ/です。 (高是名學生)
- うちは　しずかだ/です。(我家很安靜)

## 助動詞 現在 將來的句詞尾

|                 |      | 敬體 ます-形   | 簡體         | 敬體 んです-形 |
|-----------------|------|--------------|-------------|-------------|
| いく (去)(G-1)    | 肯定 | いきます (ます-形) | いき (基本形)  | いきんです       |
|                 | 否定 | いきません        | いかない (ない-形) | いかないんです     |
| くる (來)(不規則) | 肯定 | きます (ます-形)  | くる (基本形)  | くるです        |
|                 | 否定 | きません         | こない (ない-形)  | こないんです      |


|      | 敬體 普通形  | 簡體      | 敬體 んです-形  |
|------|--------------|-----------|--------------|
| 肯定 | ...です        | ...だ      | ...なんです      |
| 否定 | ...ではありません   | ...ではない   | ...ではないんです   |
|      | （...じゃありません） | （...じゃない） | （...じゃないんです） |
|      | ...ではないです    |           |              |
|      | （...じゃないです）  |           |              |

- ()內為口語形式
- ではありません(書面語) / じゃありません(口語) /  為 です 的否定形式。前者更加禮貌。

## くる (來)　的使用細節

- 說話者回到現在的位置 (但自己家不能用)
- 聽話者/第三者來到說話者現在的位置
- 聽話者/第三者來到說話者的家
- 其他情形 用いく (去)
